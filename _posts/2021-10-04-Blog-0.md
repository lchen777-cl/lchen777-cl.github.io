## Draw the scatterplot

Now we are ready to create a simple scatterplot of penguins' culmen size. We use for loop to color different species to see whether the data form some clusters. Don't forget to append legend to the graph.

```python
from matplotlib import pyplot as plt

fig, ax = plt.subplots(1)

for s in penguins['Species'].unique():
    df = penguins[penguins['Species'] == s]
    ax.scatter(df['Culmen Length (mm)'], df['Culmen Depth (mm)'], label = s)

plt.legend()
```
